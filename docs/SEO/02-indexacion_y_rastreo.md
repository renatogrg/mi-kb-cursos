# On-Page SEO

## Concepto

Optimizar elementos **dentro** de tu página para que Google la entienda.

## Checklist

- [ ] H1: 1 único por página
- [ ] Meta description: 155-160 caracteres
- [ ] URL semántica: `/blog/seo-basico` (no `/blog/123`)
- [ ] Palabra clave: 1-2% densidad

## Herramientas

- **Yoast SEO**: Plugin para WordPress
- **Google Search Console**: Herramienta oficial de Google

## Recursos

- [Google Search Central](https://developers.google.com/search)
